# 822 ordförståelsefrågor (~synonymi/definition) ur Högskoleprovet 2006--2021
# 40 frågor/prov våren 2006 -- våren 2011, 20 frågor per prov hösten 2011 -- våren 2021 (våren 2020 gavs inget prov, våren 2021 gavs två prov)
# + 2 övningsfrågor
#
# Fält (tabseparerade):
# 1: fråge-id ("h"+år+"a"|"b"|"c"("a"|"b")+frågenummer ("00" = övningsfråga)
# 2: frågeord
# 3: svarsalternativ 
# 4: facit (0=fel; 1=rätt)
# 5: kommentar: om frågan är märkt "utgår" i facit -- totalt fyra ord som läckts på internet strax före provet våren 2012)
# 
# Anpassade från högskoleprovet av Lars Borin 2015 (2006--2014), samt Yvonne Adesam 2021 (2015--2021)
# Licens: Creative Commons CC-BY 4.0 International <http://creativecommons.org/licenses/by/4.0/>
# (hänvisa till Språkbanken, Göteborgs universitet)
#
#
#
# 822 vocabulary test items (~synonymy/definition) from the Swedish Scholastic Aptitude Test 2006--2021
# 40 items/test spring 2006 -- spring 2011, 20 items/test fall 2011 -- spring 2021 (no test was given spring 2020, two tests were given spring 2021)
# + 2 practice items
#
# Fields (tab-separated):
# 1: item id ("h"+year+"a"|"b"|"c"("a"|"b")+item number ("00" is a practice item)
# 2: target item
# 3: answer alternative 
# 4: key (0=incorrect; 1=correct)
# 5: comment: whether the item is marked "excluded" in the answer key -- four items in total that were leaked on the internet immediately before the 2012 spring test)
#
